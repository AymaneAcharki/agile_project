# Portfolio Design Framework

## Visual Identity
- **Style**: Modern minimalist with editorial sophistication
- **Aesthetic**: Clean, professional, high-impact visual presentation
- **Target**: Technical recruiters and hiring managers

## Color Palette
- **Primary**: Deep charcoal (#1a1a1a) for headers and strong text
- **Secondary**: Warm gray (#6b7280) for body text and subtle elements
- **Accent**: Soft blue (#3b82f6) for interactive elements and highlights
- **Background**: Pure white (#ffffff) with subtle section dividers
- **Success**: Muted green (#10b981) for skill tags

## Typography
- **Display Font**: "Playfair Display" - Bold serif for impactful headings
- **Body Font**: "Inter" - Clean, readable sans-serif for all content
- **Code Font**: "JetBrains Mono" - For code snippets and technical content

## Visual Language
- **Grid System**: 12-column responsive grid with consistent spacing
- **Spacing**: 8px base unit system for harmonious proportions
- **Shadows**: Subtle, realistic shadows for depth without distraction
- **Borders**: Minimal 1px borders in light gray for definition

## Interactive Elements
- **Buttons**: Rounded corners (8px), subtle hover animations
- **Cards**: Clean white backgrounds with hover lift effects
- **Navigation**: Fixed header with smooth scroll behavior
- **Project Grid**: Masonry layout with filtering capabilities
